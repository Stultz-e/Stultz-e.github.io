Case Study #3

Features Build Out

Problem

The existing site has become outdated. As the site's user base has grown additional admin features became necessary to maintain and moderate user submitted content.

Tools

Chat Widget & Forms

We created a chat widget from the homepage allowing users to schedule a tour from anywhere on the marketing site. I designed the information architecture and prototyped the user flow in Adobe XD.

New Templates

Reworked many of the marketing sites using the same successful call to actions with a new modern reboot.

Floorplan Picker

We created an interactive template for specific floors of a complex. I created the vector image in illustrator and JQuery/HTML5/SCSS was used for the prototype.

Results

This sprint was a busy one. The company had a conference and wanted to show off the features in development. We added new floorplan tiles, improved the UX of forms capturing methods and the rent payment user flow.

This site gets a ton of traffic so it was important to me to do a good job. with hundreds of unique screens, writing clean reusable code was essential.
