Case Study #2

Integrating Prototyping into the Company workflow

Problem

The existing site has become outdated. As the site's user base has grown additional admin features became necessary to maintain and moderate user submitted content.

Tools

Tasks & Responsibilities

Design

Adobe XD was used for information architecture and prototyping.

Created a login modal that supports social media login & Sign up. New global site search designed to allow users to search keywords.

We needed to create an admin side of the site which allowed...

  • Dynamic trending content options
  • Make Featured (button on user created media)
  • Notification panel for user submitted content which needs moderation
  • Webcast (podcast) media player

Development

I took my design specs made while collaborating with the CEO and coded the system in SASS and html. I would later have to make revisions in the cshtml files as scope creep set in.
