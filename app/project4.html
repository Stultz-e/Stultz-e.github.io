Case Study #4

Features Build Out

Problem

The existing site has become outdated. As the site's user base has grown additional admin features became necessary to maintain and moderate user submitted content.

Tools

Company Website

I built this 5 page site using materialize a CSS framework. I added some animations, custom graphics and made it fully responsive in any screen. They use our platform for a large number of properties so this corporate site was basically a favor.

Custom Video Footage

A costumer had changed software providers for their rental properties. They came back asking us if we could make a "montage" of video clips and optimize it for web and make it the hero image for their landing page. I used After Effects for scene transitions and added an overlay.

I used Adobe Encoder to convert the AE project into an optimized MP4 (under 10MB).

Animated Logos

A company affiliate needed a new logo for their marketing revamp. They needed an animated umbrella which I designed in Illustrator, animated in After Effects and used a JavaScriptplugin that "Lottie.js"

Results
