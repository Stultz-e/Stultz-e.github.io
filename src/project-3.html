Altrac

Design System

Summary

Altrac is an app used to help observe and manage data for agricultural control equipment.

This project was actually more of a rebuild of an existing design system. Most of the components were entact. The primary objective was to convert the Figma file to a Sketch file for future designers to use.

Tools Used

Tasks & Responsibilities

The project was converting a Figma design system over to Sketch. The clients preferred the software and wanted components and new features to be available in the Sketch format. The project had over 200+ screens.

The System

Symbols, components and were built for their respective features. The idea was that someone with a base level of Sketch knowledge could jump in and start making features based on the design system. I migrated the Figma piecemeal designs and converted it to Sketch.
