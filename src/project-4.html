Tasks & Responsibilities

This project was focused on Bootstrap 4 theming. Custom animations were set for a timed flashing effect. Multiple sliders for tablet screens. The website primarily serves as a funnel for the app which uses demographic data for likely voters in county districts.

The Sketch files were exported to Zeplin and the mockups/assets were exported at 2x size for retina display. This site was a developed 'mobile first' and The components vary a great deal between desktop, tablet and mobile.
